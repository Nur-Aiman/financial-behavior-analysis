# Financial Behavior Analysis

A comprehensive full-stack application for tracking and analyzing financial behavior patterns.

## Project Structure

- **FBA_BE**: Node.js/Express backend server
- **FBA_FE**: React frontend application
- **db_backup**: Database backups and migrations

## Tech Stack

### Backend
- Node.js with Express.js
- PostgreSQL/MySQL database
- Knex.js for database migrations

### Frontend
- React.js
- CSS for styling
- RESTful API integration

## Setup Instructions

### Backend Setup
```bash
cd FBA_BE
npm install
npm start
```

### Frontend Setup
```bash
cd FBA_FE
npm install
npm start
```

### Development Mode
```bash
npm run dev
```

## Database

Database files and backups are stored in the `db_backup/` directory.

## Environment Variables

Create a `.env` file in the `FBA_BE` directory with your database and configuration credentials.
